# MoneyFloof

## Description
**MoneyFloof** is a monthly-oriented **Kanban expense manager**, customizable and focused on calculations and visualizations.  
It helps users track their online orders and expenses more effectively than Trello or a simple spreadsheet by providing automatic calculations and visual insights.

**Goal:**  
Create a practical tool to track spending, estimate savings, and visualize monthly budgets.

**Target audience:**  
Anyone who wants to manage their expenses and online orders more efficiently.

## Tech Stack
- **Backend:** PHP 8.2 with Slim 4  
- **Frontend:** React + Styled Components  
- **Database:** MySQL  
- **Main Libraries:** WIP  
- **Why PHP?** Limitation of the IONOS hosting plan  
- **Future version:** Planned migration to Rust for a more modern and performant backend
